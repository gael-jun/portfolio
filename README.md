## 🚀 Démarrer en local

Prérequis: Node.js 18+ et npm.

1. Installer les dépendances
	- `npm install`
2. Lancer le serveur de dev
	- `npm run dev`
3. Ouvrir le site
	- http://localhost:3000

Build & production:
- `npm run build` puis `npm start`
